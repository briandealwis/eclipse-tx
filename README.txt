This package is a set of Eclipse plugins that I've written over the
years to help build and code in Eclipse. 

ca.mt.wb.spies: a set of spies into the workings of the Eclipse Workbench.
  It provides a Selection view, showing details about the current
  selection in the workbench.  It also provides event loggers that
  dump SWT events and Command execution events to stdout, toggled
  through the Window menu.

ca.mt.wb.jdt: a set of JDT tools
  It provides a slimmed down Java Browsing view, and two new search
  dialogs, Open Field and Open Method, like Open Type
